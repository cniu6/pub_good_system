## 📚 核心模块

### 1. 请求系统

**关键文件**:
- `src/service/http/` - Alova 请求封装
- `src/service/api/user/` - 用户 API（登录、个人中心）
- `src/service/api/admin/` - 管理端 API（懒加载）
- `src/service/api/` - 系统级 API

**使用方式**:
```typescript
import { fetchLogin, fetchGetUserProfile } from '@/service'

// 登录
const res = await fetchLogin({ username, password })

// 获取用户信息
const user = await fetchGetUserProfile()
```

📖 **详细文档**: [../doc/前端请求.md](../doc/前端请求.md)

### 2. 路由系统

**关键文件**:
- `src/router/index.ts` - 路由配置

**使用方式**:
```typescript
import { useRouter } from 'vue-router'

const router = useRouter()

// 跳转
router.push('/dashboard')

// 获取参数
const id = router.currentRoute.value.params.id
```

### 3. 状态管理

**关键文件**:
- `src/store/auth.ts` - 认证状态

**使用方式**:
```typescript
import { useAuthStore } from '@/store'

const authStore = useAuthStore()

// 登录
await authStore.login({ username, password })

// 获取用户信息
console.log(authStore.userInfo)

// 登出
authStore.logout()
```

### 4. 国际化

**关键文件**:
- `src/locales/*.json` - 语言文件

**使用方式**:
```vue
<template>
  <div>{{ $t('login.title') }}</div>
</template>

<script setup>
import { useI18n } from 'vue-i18n'

const { t } = useI18n()

console.log(t('login.title'))
</script>
```

### 5. 类型定义

**关键文件**:
- `src/typings/entities/*.d.ts` - 实体类型

**使用方式**:
```typescript
// 使用预定义类型
const user = ref<Entity.User | null>(null)

// 扩展类型
declare namespace Entity {
  interface NewEntity {
    id: number
    name: string
  }
}
```

## 📝 开发规范

### 1. 组件命名

```vue
<!-- ✅ 推荐 -->
<UserProfile />
<UserList />

<!-- ❌ 不推荐 -->
<user-profile />
<userprofile />
```

### 2. API 命名

```typescript
// ✅ 推荐
fetchGetUserList
fetchCreateUser
fetchUpdateUser

// ❌ 不推荐
getUserList
getUser
requestUserList
```

### 3. 类型命名

```typescript
// ✅ 推荐
interface UserInfo { }
type UserRole = 'admin' | 'user'

// ❌ 不推荐
interface userInfo { }
type userrole = string
```

## ⚠️ 开发注意事项

1. **使用别名**: 使用 `@/` 别名导入模块
2. **类型安全**: 为所有变量和函数添加类型
3. **错误处理**: 使用 try-catch 处理异步操作
4. **组件拆分**: 大组件拆分为小组件
5. **代码复用**: 使用 composables 复用逻辑

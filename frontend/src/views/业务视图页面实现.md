# 业务视图页面实现 (Views) 深度流证

## 简介
Views 目录是系统的功能展示层，基于 Naive UI 构筑，涵盖了从基础权限到复杂业务场景的完整实现。

## 详细页面流证

### 1. 内置基础页面 (Build-in)
- **Login (登录页)**:
    - **Login**: 核心表单，集成极验滑动验证。
    - **Register**: 用户自助注册，包含基础字段校验。
    - **ResetPwd**: 密码重置流程。
- **UserCenter (个人中心)**: 展示当前用户的详细资料、修改头像及安全设置。
- **Not-found (404)**: 优雅的错误引导页。

### 2. 仪表盘与监控 (Dashboard)
- **Workbench (工作台)**:
    - 聚合展示系统公告、快捷入口、待办事项。
    - 使用 `chart.vue` 展示基础业务走势。
- **Monitor (监控页)**:
    - 实时系统资源监控。
    - 采用多维度图表 (`chart.vue`, `chart2.vue`, `chart3.vue`) 渲染 CPU、内存、请求量等关键指标。

### 3. 系统配置 (Setting)
- **Account (账户管理)**:
    - 提供用户列表的分页展示、搜索、新增、编辑及状态切换。
    - **TableModal**: 抽离的账户编辑弹窗。
- **Dictionary (字典管理)**:
    - 管理系统内的枚举值（如性别、状态、分类）。
    - 支持树形结构展示及编码唯一性校验。
- **Menu (菜单管理)**:
    - 动态配置系统侧边栏、路由及权限标识。
    - 支持图标选择、排序、隐藏等高级配置。

### 4. 演示与示例 (Demo)
- **Fetch (网络请求演示)**:
    - ** refreshToken**: 演示 Token 过期后自动刷新的无感体验。
    - ** Transform**: 演示 Alova 数据转换器的用法。
- **Editor (编辑器演示)**:
    - 集成 Markdown 和富文本编辑器，支持图片上传。
- **List (列表演示)**:
    - 包含普通表格、卡片布局及支持拖拽排序的复杂列表。
- **Permission (权限演示)**:
    - 演示基于指令 `v-permission` 和函数 `usePermission` 的按钮级权限控制。

## 技术实现要点
- **组合式函数驱动**: 复杂逻辑（如表格分页、图表自适应）均封装在 `hooks` 中。
- **类型安全**: 所有页面均引入 `typings` 中的实体定义，确保数据渲染无误。
- **响应式适配**: 布局组件自动感应 `isMobile` 状态，动态切换展示模式。

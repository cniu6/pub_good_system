## 🎯 项目简介

FST (Full Stack Template) 是一个基于 Go (Gin) 和 Vue 3 的全栈后台管理系统模板，采用前后端分离架构，支持插件化扩展。

### 技术栈

- **后端**: Go + Gin + MySQL + JWT
- **前端**: Vue 3 + TypeScript + Vite + Naive UI + Alova
- **构建**: 支持 Windows/Linux, amd64/arm64

## 🚀 快速开始

### 环境要求

- Go 1.24+
- Node.js 18+ & pnpm
- MySQL 8.0+

### 启动开发环境

```bash
# Windows
./dev.bat

# 或手动启动
# 终端 1: 后端
go run backend/cmd/main.go

# 终端 2: 前端
cd frontend && pnpm dev
```

### 生产构建

```bash
# Windows
./build.bat

# 构建产物在 build/ 目录
```

## 🛠️ 常用命令

### 后端命令

```bash
# 运行
go run backend/cmd/main.go

# 构建
go build -o fst_backend.exe backend/cmd/main.go

# 更新 Swagger
go run github.com/swaggo/swag/cmd/swag init -g backend/cmd/main.go -o backend/docs

# 测试
go test ./backend/...
```

### 前端命令

```bash
cd frontend

# 安装依赖
pnpm install

# 开发
pnpm dev

# 构建
pnpm build

# 类型检查
pnpm typecheck

# 代码检查
pnpm lint
```

## ⚙️ 环境变量

### 必需配置

```env
# 数据库
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=fst_platform

# JWT
JWT_SECRET=your-secret-key
```

```env
# SMTP
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=your-email@gmail.com
SMTP_PASSWORD=your-app-password

# 极验
GEETEST_ENABLED=false

GEETEST_KEY=your-geetest-key

```

## 🎯 AI 助手使用指南

### 需要了解的内容

1. **发送邮件**: 阅读 [doc/邮件系统.md](doc/邮件系统.md)
2. **用户认证**: 阅读 [doc/JWT认证.md](doc/JWT认证.md)
3. **数据库操作**: 阅读 [doc/数据库模型.md](doc/数据库模型.md)
4. **开发插件**: 阅读 [doc/插件系统.md](doc/插件系统.md)
5. **API开发**: 阅读 [doc/API路由.md](doc/API路由.md)
6. **前端请求**: 阅读 [doc/前端请求.md](doc/前端请求.md)

### 常见任务

#### 添加新 API

1. 在 `backend/app/controllers/` 创建或修改控制器
2. 在 `backend/routes/routes.go` 添加路由
3. 在 `frontend/src/service/api/` 添加 API 函数

#### 发送邮件

1. 确保 `.env` 配置了 SMTP
2. 使用 `utils.SendEmail()` 或模板系统
3. 异步记录邮件日志

#### 开发插件

1. 在 `backend/app/plugins/` 创建插件目录
2. 实现 `Plugin` 接口
3. 在 `main.go` 注册插件
